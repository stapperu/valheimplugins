# Boss Summons Allies

## Why This Mod?

If you're looking to spice up boss encounters by adding a level of challenge.

This mod aims to give the bosses the ability to summon waves of enemies which are typically associated when you are raided.


If you're interested in a further challenge to the bosses, check out my other Boss related mod: 

- Severon-BossInvokesForsakenPowers

## Features

The enemies spawned are based on their Raid Event associated with them.

For example:
- Eikthyr will summon Boars and Necks. 
- The Elder summons the Greydwarves, Brute and Shamans.
- Bonemass will summon Draugr and Skeletons.
- Moder will summon Drakes.
- Yagluth will summon Fulings, Berserkers and Shamans.
- Fader will summon Charred.

- The Queen is the exception to this as her encounter is indoors. At some point I will work on a workaround, however because I hook into the raid event system, it won't spawn Seekers inside a dungeon. (Lucky for you!)

Time between waves can be configured per boss. This duration is reduced by 5% per additional player down to 60 seconds for 10 players at the default setting.

The default time has been tested for balance to allow time to clear the wave and time on the boss before the next boss. However further balance and testing is still ongoing.

## Manual Install

Install \[BepInEx for Valheim](https://valheim.thunderstore.io/package/denikson/BepInExPack\_Valheim/).

Download the mod. Extract the zip folder and place the extracted folder in `BepInEx/plugins` folder.
